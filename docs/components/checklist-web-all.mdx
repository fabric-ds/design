# General checklist for every web page on FINN

<TableOfContents />

## 1. Document title
Make sure the title element in the HTML head follows the title tag guidelines.

## 2. Main element
Every page must have exactly one main element. Generally this element should wrap all content between Header and Footer. It must never include Header or Footer.

## 3. H1 element
Every page must have at least one main heading (h1 element) located inside the main element mentioned in ยง2. It can be hidden visually with sr-only class if necessary.

## 4. Heading structure
Make sure all headings on the page follow the [Heading guidelines for screen readers](/screenreaders#headings).
