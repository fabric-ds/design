### Update Express layout constructor

When making the transition to Fabric, we highly recommend making sure that no
Troika files are being included in the page. If your app is using the
`@finn-no/express-layout` module, you can disable Troika file inclusion via a
constructor flag.

First, ensure Express layout is installed at the latest version.

```sh
npm install @finn-no/express-layout@latest
```

Next, set the `troika` option to `false` in the module constructor.

```js
import ExpressLayout from '@finn-no/express-layout';

new ExpressLayout({
  // ...
  troika: false,
  // ...
});
```
